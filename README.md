## 介绍
高效脚手架一枚，欢迎使用~
如果报错的话 请提个issues~
另外服务端用的是egg搭建  集成socket,redis（功能还在完善~）代码（https://github.com/Murrayee/m-server）

## start
- yarn install （尽量用yarn 安装 ，npm有可能会出问题 ）
- yarn run ios (android)
